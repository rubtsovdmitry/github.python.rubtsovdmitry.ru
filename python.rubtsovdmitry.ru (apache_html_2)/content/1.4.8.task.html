8. Количество и сумма цифр в числе (в) (в коде есть строки из следующих тем: "строки").

Условие:
Пользователь вводит большое целое число (например, 4568). Программа считает сумму цифр в этом числе (например, 4+5+6+8 для числа 4568).
Код: 
number = input("Введите большоее целое число: ")

amount = 0
for i in number:
    amount += int(i)
    
print("Сумма цифр в этом числе равно", amount)
