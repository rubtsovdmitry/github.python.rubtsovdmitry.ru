6. Количество и сумма цифр в числе (а) (в коде есть строки из следующих тем: "строки").

Условие:
Пользователь вводит большое целое число (например, 4568).Программа выводит на экран все цифры этого числа по одной (в любом порядке).
Код: 
number = input("Введите большоее целое число: ")
for i in number:
    print(i)
