10. Текущее время.

Условие:
Модуль time в Python включает в себя несколько очень полезных функций для работы со временем. Одна из таких функций – asctime – считывает текущее системное время компьютера и возвращает его в удобном для восприятия виде. Используйте эту функцию для отображения на экране текущей даты и времени.
Код: 
from time import asctime                # загружает функцию текущего времени из модуля time

print("Текущее время\n")
print(asctime())
