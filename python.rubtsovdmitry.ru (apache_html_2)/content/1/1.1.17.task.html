17. Площадь и объем (в коде есть строки из следующих тем: "библиотеки").

(открыть в новой вкладке)
Условие:
Напишите программу, которая будет запрашивать у пользователя радиус и сохранять его в переменной r. После этого она должна вычислить площадь круга с заданным радиусом и объем шара с тем же радиусом. Используйте в своих вычислениях константу pi из библиотеки math.
Площадь круга вычисляется по формуле area = π * (r ** 2), а объем шара – по формуле volume = 4 / 3 * π * (r ** 3).
Код: 
from math import pi                         # подключение дополнительной константы из библиотеки math
print()

r = float(input("Введите радиус: "))        # ввод радиуса без единиц измерения                                                                                                

area = pi * r ** 2                          # формула для площади круга
volume = 4 / 3 * pi * r ** 3                # формула для объёма шара

print(area)
print(volume)
