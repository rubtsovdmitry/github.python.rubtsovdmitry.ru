Условие:
Пользователь вводит температуру в цельсиях в формате xxx.xx, программа её переводит в Фаренгейты.
Формула для перевода: t F  = (t C  × 9/5) + 32
Код: 
temperature = float(input("Введите температуру в цельсиях: "))
formula = (temperature * 9 / 5) + 32

print("Температура в фаренгейтах", round(formula, 2))
