(открыть в новой вкладке)
Условие:
На игровой рулетке в казино 38 выемок для шарика: 18 красных, столько же черных и две зеленые, пронумерованные 0 и 00 соответственно. Красные числа на рулетке: 1, 3, 5, 7, 9, 12, 14, 16, 18, 19, 21, 23, 25, 27, 30 32, 34 и 36. Остальные номера в диапазоне положительных чисел до 36 – черные. Игрок может сделать самые разные ставки. Мы в данном упражнении будем рассматривать лишь следующие виды ставок:
- одно число (от одного до 36, а также 0 или 00);
- красное или черное;
- четное или нечетное (заметьте, что номера 0 и 00 не являются ни четными, ни нечетными).
Напишите программу, имитирующую игру на рулетке при помощи генератора случайных чисел в Python. После запуска рулетки выведите на экран выпавший номер. Нужно придумать коэффициенты для выигрышных сделок.

forecast = input("%100s" % "красное / чёрное / чётное / нечётное / (цифра от 0 до 36): ")
x = random.randint(0,37)
print()
print("%99s" % "Проверка", x)

# запуск цикла
while score > 0 and stavka <= score: 
    if forecast == "красное" and x in COLOUR["red"]:
        print("%99s" % "Выигрыш")
        score = round((score + stavka * 0.9), 2)
    elif forecast == "чёрное" and x in COLOUR["black"]:
        print("%99s" % "Выигрыш")
        score = round((score + stavka * 0.9), 2)

        print("%99s" % "Выигрыш")

        print("%99s" % "Выигрыш")
