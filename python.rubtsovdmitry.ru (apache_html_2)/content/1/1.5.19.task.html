19. Вывести случайный элемент из списка (в коде есть строки из следующих тем: "библиотеки").

(открыть в новой вкладке)
Условие:
Вывести случайный элемент из списка.
Пример списка: a = ["x", "y", "z", "a", "b", "c"]
Код: 
import random

a = ["x", "y", "z", "a", "b", "c"]
print(a[random.randint(0, len(a))])
