1. Задачи на циклы for.

Условие:
Вывести на экран все числа от 25 до 70 не включительно, посчитать их сумму.
Код: 
# первый вариант

diapazon = range(25, 70)
for i in diapazon:
    print(i)
    
print()
print("Сумма всех чисел дипазона равна:", sum(diapazon))
		
# второй вариант

diapazon = range(25, 70)
amount = 0

for i in diapazon:
    amount += i
    print(i)
    
print()
print("Сумма всех чисел дипазона равна:", amount)
