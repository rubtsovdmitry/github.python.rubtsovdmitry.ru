20. Свободное падение (в коде есть строки из следующих тем: "модули").

(открыть в новой вкладке)
Условие:
Напишите программу для расчета скорости объекта во время его соприкосновения с землей.
Пользователь должен задать высоту в метрах, с которой объект будет отпущен и начальную скорость. Предположим, что ускорение свободного падения равно 9,8 м/с2. При известных начальной скорости (vi), ускорении (a) и дистанции (d) можно вычислить скорость при соприкосновении объекта с землей по формуле:
V = Корень квадратный из (vi ** 2 + 2 * a * d)
Код: 
from math import sqrt

A = 9.8                                                                                       # ускорение свободного падения
v = float(input("%70s" % "Введите начальную скорость в м/c: "))                
d = float(input("%70s" % "Введите дистанцию до Земли в м: "))

result = sqrt(v ** 2 + 2 * A * d)

print("%69s" % "Скорость объекта во время соприкосновения с Землёй", round(result), "м/c")
print("%69s" % "или", round(result / 1000 * 3600), "км/час")
