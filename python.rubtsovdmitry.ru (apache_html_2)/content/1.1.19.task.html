19. Объем цилиндра (в коде есть строки из следующих тем: "модули").

Условие:
Объем цилиндра может быть рассчитан путем умножения площади круга, лежащего в его основе, на высоту.
Напишите программу, в которой пользователь будет задавать радиус цилиндра и его высоту, а в ответ получать его объем, округленный до одного знака после запятой.
Код: 
from math import pi

radius = float(input("%40s" % "Введите радиус цилиндра в см.: "))
height = float(input("%40s" % "Введите высоту цилиндра в см.: "))

area = pi * radius ** 2
v = area * height

print("%39s" % "Объём цилиндра:", round(v, 1), "см^3")
