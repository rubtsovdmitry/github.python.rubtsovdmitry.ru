2. Задачи на циклы for (в коде есть строки из следующих тем: "списки").

Условие:
Вывести все числа от 20 до 100, которые делятся на 5.
Код: 
# первый вариант

diapazon = range(20, 101)
for i in diapazon:
    if i % 5 == 0:
        print(i)
		
# второй вариант

diapazon = range(20, 101)
result = [i for i in diapazon if i % 5 == 0]    # пока ещё темы списков не было, забегая вперёд, вот такое решение (для списков своя запись для циклов с условием)
print(result)
