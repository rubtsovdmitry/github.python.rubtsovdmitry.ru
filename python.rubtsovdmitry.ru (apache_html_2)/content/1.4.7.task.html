7. Количество и сумма цифр в числе (б) (в коде есть строки из следующих тем: "строки").

Условие:
Пользователь вводит большое целое число (например, 4568). Программа считает количество цифр в этом числе.
Код: 
number = input("Введите большоее целое число: ")

count = 0
for i in number:
    count += 1
print("Количество цифр в этом числе равно", count)
