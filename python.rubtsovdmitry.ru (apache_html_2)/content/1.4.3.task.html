3. Задачи на циклы for (в коде есть строки из следующих тем: "списки").

(открыть в новой вкладке)
Условие:
Вывести все числа от 20 до 100, которые делятся на 7 или на 9.
Код: 
# первый вариант

diapazon = range(20, 101)
for i in diapazon:
    if i % 7 == 0 or i % 9 == 0:
        print(i)
      
# второй вариант

diapazon = range(20, 101)
result = [i for i in diapazon if i % 7 == 0 or i % 9 == 0]    # для списков своя запись для циклов с условием
print(result)
