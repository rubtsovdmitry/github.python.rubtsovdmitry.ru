4. Чет или нечет?

(открыть в новой вкладке)
Условие:
Напишите программу, запрашивающую у пользователя целое число и выводящую на экран информацию о том, является введенное число четным или нечетным.
Код: 
number = int(input("%33s" % "Введите целое число: "))

if number == 0:
    print("%32s" % "Ваше число:", "ноль")
elif number % 2 == 0:
    print("%32s" % "Ваше число:", "чётное")
else:
    print("%32s" % "Ваше число:", "нечётное")
