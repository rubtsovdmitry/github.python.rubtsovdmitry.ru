7. Кредитные карты.

(открыть в новой вкладке)
Условие:
Программа должна открыть файл "Список кредитных карт.txt". Нужно вывести информация по людям на экран, на котором номера карт будут отображены в виде "****************".
⇒ Список кредитных карт
Код: 
import re

buf = open("./Список кредитных карт.txt", "r")
s = buf.read()

regex_obj = re.compile(r"\d{4} \d{4} \d{4} \d{4} ?\d?")                   
s = regex_obj.sub("<credit card> **************", s)
regex_obj = re.compile(r"\d{16}(\d{3})?")                   
s = regex_obj.sub("<credit card> **************", s)

print(s)
