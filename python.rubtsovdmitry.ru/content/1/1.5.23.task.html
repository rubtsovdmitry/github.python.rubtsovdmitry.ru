23. Порядок сортировки.

(открыть в новой вкладке)
Условие:
Напишите программу, которая будет запрашивать у пользователя целочисленные значения и сохранять их в виде списка. Индикатором окончания ввода значений должна служить пустая строка. Затем программа должна вывести на экран все введенные пользователем числа в порядке возрастания.
Код: 
my_list = []
while True:
    digit = input("%100s" % "Введите число для заполнения списка (для отмены просто нажмите <Enter>): ")
    if digit == "":
        break
    my_list.append(int(digit))
my_list.sort()
print("Числа в порядке возрастания: ", end="")
for i in my_list:
    print(i, " ", end="")
print()
