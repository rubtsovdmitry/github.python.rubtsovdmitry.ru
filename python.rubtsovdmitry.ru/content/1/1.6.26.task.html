26. Поиск по значению.

(открыть в новой вкладке)
Условие:
Дан словарь: my_dict = { "a": 1, "b": 2, "c": 3, "d": 4, "e": 5, "f": 1, "g": 1}
Нужно найти все ключи со значением 1.
Код: 
my_dict = { "a": 1, "b": 2, "c": 3, "d": 4, "e": 5, "f": 1, "g": 1}
result = [i for i in my_dict if my_dict[i] == 1]
print(result)
