25. Является ли строка целым числом?

(открыть в новой вкладке)
Условие:
Нужно написать функцию, которая проверяет является ли строка целым числом. Если есть пробелы в начале и в конце строки, программа должна это как-то обработать.
Код: 
def my_function(x):
    x = x.strip()
    if x.isdigit():
        return "Это цифра."
    else:
        return "Это не цифра."


x = input("%80s" % "Введите символы и программа проверит цифры это или нет: ")
print("%79s" %  my_function(x))
